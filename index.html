Lançamento de Projéteis - Uma Análise Computacional

O objetivo principal deste projeto é realizar uma análise computacional do lançamento de projéteis, aplicando conceitos fundamentais da disciplina de Física I. Ele visa aprimorar a compreensão dos conceitos teóricos por meio de uma abordagem prática e visual. O projeto oferece uma ferramenta interativa e educativa para explorar e compreender os princípios do lançamento de projéteis. Antes de apresentarmos a ferramenta, compartilhamos um breve overview do desenvolvimento do projeto e dos desafios enfrentados.




O DESENVOLVIMENTO

Os colaboradores encarregados da estrutura e implementação do projeto em programação conduziram uma consulta com o intuito de identificar a abordagem mais apropriada para abordar o desafio proposto pelo professor. Após uma análise detalhada, chegaram a um consenso de que a componente de cálculo deveria ser implementada utilizando JavaScript, enquanto a parte relacionada à interface web e modelagem seria desenvolvida por meio de HTML e CSS.
